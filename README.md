# Palmer Penguins: Exploratory Data Analysis

## Overview
This was an attempt at Exploratory Data Analysis (EDA) project using Python. The goal of this project was to practice loading data, cleaning it, and creating basic visualizations to find patterns.

## The Dataset
I used the **Palmer Penguins** dataset, which contained size measurements for three penguin species observed on three islands in the Palmer Archipelago, Antarctica.

## Tools Used
* **Python**
* **Pandas** (Data manipulation)
* **Seaborn & Matplotlib** (Data visualization)

## Findings
* Adelie is the most common penguin species in this dataset.
* There is a strong positive correlation between a penguin's flipper length and its body mass.
* Gentoo penguins only live on Biscoe island.
* Chinstrap penguins only live on Dream island.
